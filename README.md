# ASP.NET Core Web API

Exercício simples de criação de uma Web API desenvolvida com ASP.NET Core (https://learn.microsoft.com/pt-pt/training/modules/build-web-api-aspnet-core/).

## 📌 Sobre

Este projeto tem como objetivo praticar a criação de uma API REST utilizando ASP.NET Core.

A aplicação implementa operações básicas de CRUD e simula o gerenciamento de dados em memória, sem utilização de banco de dados.

---

## 🚀 Tecnologias

- ASP.NET Core  
- C#  
- .NET  
- Swagger  

---

## ⚙️ Funcionalidades

- Listar registros  
- Buscar registro por ID  
- Criar novo registro  
- Atualizar registro existente  
- Remover registro  

---

## ▶️ Como executar

```bash
dotnet restore
dotnet run
